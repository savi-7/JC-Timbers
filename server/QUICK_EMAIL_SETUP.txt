📧 QUICK EMAIL SETUP FOR JC TIMBERS
=====================================

⚠️ IMPORTANT: Update your server/.env file with email credentials

Add these lines to server/.env:

EMAIL_HOST=smtp.gmail.com
EMAIL_PORT=587
EMAIL_USER=jctimbers@gmail.com
EMAIL_PASSWORD=your-gmail-app-password-here

📝 HOW TO GET GMAIL APP PASSWORD:
1. Go to https://myaccount.google.com/
2. Security → 2-Step Verification (turn it ON)
3. Go to https://myaccount.google.com/apppasswords
4. Generate app password for "Mail" → "Other (JC Timbers)"
5. Copy the 16-character password
6. Paste it in EMAIL_PASSWORD above
